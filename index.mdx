## O que é o Affiliatus?

O **Affiliatus** é uma plataforma completa para gestão de programas de afiliados voltada para empresas SaaS. Crie campanhas, rastreie performance em tempo real, gerencie comissões e processe pagamentos de forma automatizada através de uma integração plug-and-play.

</Card>
